# mad-coffee-menu

This repository contains a set of JSON data and relevant accompanying assets intended to spoof the contents of a menu from a coffee shop. This data is intended purely for use within the context of a Mobile App Development class assignment with Staffordshire University in order to provide an external data source for an app to read from - **Please do not send pull requests to this repository!**
